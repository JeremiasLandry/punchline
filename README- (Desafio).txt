# Punchline - Tienda de Ropa Urbana

Punchline es un e-commerce que vende ropa urbana de estilo callejero.

Cuenta con

-  filtro por categorías: tomando las categorías de la base de datos se generan filtros de productos.
- vistas de listas de productos: vista general de productos con foto y detalles mínimos como el nombre y precio.
- vista de detalles: vista con todos los detalles del producto y seleccion de talle y cantidad.
- paginacion: la pagina cuenta con paginacion que permite dividir el contenido del catalogo y navegar entre
diferentes paginas.
- carrito: permite almacenar los productos agregados por el usuario.
- checkout: valida la compra del usuario y crea la orden, que luego es enviada a una base de datos de firebase.



## Instalación

1. Descargar el repositorio

2. Parado en la raíz del proyecto corré el comando 

   ```
   npm install
   ```

    para instalar todas las dependecias del proyecto

3. Usá 

   ```
   npm start
   ```

    para correr el proyecto, que estará disponible en http://localhost:3000



## Dependencias

Uso de Bootstrap para dar formato a la aplicación.

Uso de Popper JS para dar funcionalidad a componentes de Bootstrap.

Uso de React Router Dom para la navegación

Uso de FontAwesome para la utilizacion de diferentes iconos.

Uso de Firebase para el almacenamiento de productos, ordenes e imagenes.
